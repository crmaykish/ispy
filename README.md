#iSpy: Double-agent Tracking

![screenshot](screenshot.PNG)

<h2>Languages and Frameworks</h2>
 - Grails 3.0.8
 - GroovyCSV 0.2
 - Leaflet 0.7.5
 - Bootstrap 3.3.5
 - jQuery 2.1.4
 
<h2>How To Run</h2>

Requires Git and Grails.
```
git clone https://github.com/crmaykish/ispy
cd ispy
grails run-app
```
Open browser to http://localhost:8080/
